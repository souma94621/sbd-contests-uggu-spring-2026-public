<a id="c19"></a>
### C19 — Разнесение доменов по процессам
**Макс.** 3. **Артефакты:** [`src_solution/`](../src_solution/).  

**Значимо для задачи не одно только разнесение модулей по каталогам `abu/tcb` и `abu/other` (это вспомогательно для покрытия), а наличие разделения доменов безопасности по процессам (`multiprocessing`, `subprocess` и т. п.) во время выполнения. При отсутствии процессной изоляции автоматический комментарий фиксирует необходимость рассматривать кодовую базу как единую границу доверия (повышенные ожидания к покрытию и SBOM).**

**Уровни:** эвристика по признакам процессов (`multiprocessing`/...), по доменному коду (`domains`, `monitor`, `request/response`).

Если в решении **нет** одновременно каталогов `abu/tcb` и `abu/other`, в комментарии также указывается трактовка Регулятора по покрытию.
