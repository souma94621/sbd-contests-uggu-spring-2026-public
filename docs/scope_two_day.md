# Упрощения объёма (ориентир «два дня — один человек»)

Актуальный практический маршрут перенесён в [quickstart_2days.md](quickstart_2days.md).  
Этот файл сохранён как справочное пояснение ограничений.

- Зависимости проекта задаются **[Pipfile](../Pipfile)** (и **Pipfile.lock**); установка только через **Pipenv** в **`.venv/`**, не в системный Python ([quality_requirements.md](quality_requirements.md)). В каталоге `src_starting_point/` по-прежнему есть `requirements.txt` для состава сертификационного пакета АБУ.
- Без Kubernetes и без TLS между сервисами в прототипе v1.
- Регулятор хранит заявки и сертификаты в памяти процесса.
- Песочница сертификации — локальный `venv` и подпроцесс `pytest` по распакованному пакету.
- SBOM заготовки — каталог [`src_starting_point/sbom/`](../src_starting_point/sbom/).
- Интеграционные тесты могут использовать `TestClient` (ASGI) для Регулятора там, где важна скорость; сценарий сертификации дублируется через HTTP к поднятому процессу при необходимости.
