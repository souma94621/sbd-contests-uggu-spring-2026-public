# Пакет документации (LaTeX)

Файл `uggu_sbd_contest_2026.tex` объединяет описание конкурсного задания, архитектуру, сертификацию, инфраструктуру, шаги, критерии оценки, список источников, FAQ и **приложения** (полный перечень C01–C22, пример отчёта, полный текст регламента, шаблон и апелляция). Версия документа задаётся в шапке `uggu_sbd_contest_2026.tex` (`\docversion`; актуально **1.7.7**). Расшифровка уровней 0–3 в Markdown: [../criteria_rubric.md](../criteria_rubric.md).

## Сборка

Из корня репозитория (опционально, для PNG диаграмм):

```bash
make diagrams
```

Сборка PDF:

```bash
cd docs/bundle
chmod +x build.sh
./build.sh
```

Либо вручную: `latexmk -xelatex uggu_sbd_contest_2026.tex`

Требуется **XeLaTeX** и шрифты **DejaVu** (или замените шрифты в `preamble.tex`). Для автоматического цикла компиляций удобно установить **latexmk**.

Результат: `uggu_sbd_contest_2026.pdf`.

## Проверка символов

После сборки откройте `uggu_sbd_contest_2026.pdf` и проверьте кириллицу, латиницу, математику в формуле нормализации, моноширинный текст и кликабельные URL.
